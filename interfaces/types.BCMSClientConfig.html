Optional enableCache

enableCache?: boolean
  • Defined in types/config.ts:21

Use memory caching. If enabled, all responses from the BCMS will be saved in memory and updated only if the data changes in the BCMS (using sockets).

IMPORTANT: This option should be enabled only on server side because a BCMS cannot handle infinite number of socket connections.
